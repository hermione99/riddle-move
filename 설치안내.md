# 📖 리마커블 무브에서 '톰 리들의 일기장(riddle)' 한글로 쓰기 — 설치 안내

펜으로 일기장에 글을 쓰면, 페이지가 **잉크를 삼키고** 잠시 뒤 **손글씨로 답장**이 써지는 앱이에요.
이 버전은 **리마커블 페이퍼 프로 무브(Move)** 전용으로 개조했고, **한글로 쓰면 한글로, 영어로 쓰면 영어로** 답합니다.

> 원작: [MaximeRivest/riddle](https://github.com/MaximeRivest/riddle) · Move/한글 개조판: [hermione99/riddle-move](https://github.com/hermione99/riddle-move) (MIT 라이선스)

---

## ⚠️ 시작 전에 꼭 읽어주세요

- **비공식 도구예요.** 리마커블 공식 앱이 아니며, 설치·사용에 따른 문제는 본인 책임입니다.
- **개발자 모드를 켜면 기기가 초기화돼요.** 아래 1단계에서 개발자 모드를 처음 켜는 분은 **먼저 클라우드 동기화**로 노트를 백업하세요.
- **OpenAI API 키(유료)** 가 필요해요. 답장 1번에 몇 원 수준으로 저렴하지만, 결제수단이 등록된 OpenAI 계정이 있어야 합니다.
- 안전장치가 있어요: 앱을 빠져나오면(다섯 손가락 탭 / 전원 버튼) 원래 화면으로 자동 복구됩니다.

---

## 🧰 준비물

1. **리마커블 페이퍼 프로 무브** 기기
2. **USB-C 케이블** (기기 ↔ 컴퓨터 연결)
3. **컴퓨터** (맥 / 윈도우 / 리눅스)
4. **OpenAI API 키** (아래 3단계에서 발급)

---

## 1단계. 개발자 모드 + AppLoad 설치 (한 번만)

이 앱은 **xovi/AppLoad**(리마커블용 앱 실행기) 위에서 돌아갑니다. 아직 안 깔았다면
**remagic** 도구가 개발자 모드부터 AppLoad까지 명령어 한 줄로 설치해줘요.

👉 설치 방법: **https://github.com/maximerivest/remagic** 의 안내를 따라주세요.

> 이미 KOReader 같은 걸 AppLoad로 쓰고 계시면 이 단계는 건너뛰어도 됩니다.

설치가 끝나면 기기 홈에 **AppLoad** 런처가 보입니다.

---

## 2단계. 앱 다운로드

아래 링크에서 zip을 받아 압축을 풀어주세요.

👉 **https://github.com/hermione99/riddle-move/releases/latest**

- 한글 버전: **`riddle-move-ko.zip`** ← 추천
- 영어 버전: `riddle-move-en.zip`

압축을 풀면 `riddle-move-ko` 라는 **폴더**가 생깁니다.

---

## 3단계. OpenAI API 키 발급 & 넣기

1. https://platform.openai.com/api-keys 에서 **Create new secret key** 로 키를 만들어요
   (키는 `sk-` 로 시작하는 긴 문자열). 결제수단이 없으면 Billing에서 소액 충전해두세요.
2. 방금 푼 폴더 안의 **`oracle.env`** 파일을 **메모장(윈도우) / 텍스트편집기(맥)** 로 엽니다.
3. 이 줄을 찾아서
   ```
   RIDDLE_OPENAI_KEY=sk-여기에_본인_키를_붙여넣으세요
   ```
   `sk-...` 부분을 **본인 키로 바꿔** 저장하세요. (따옴표 없이, 앞뒤 공백 없이 붙여넣기)

> 💡 키는 컴퓨터에서 미리 넣어두는 게 편해요. 그래야 기기에서 따로 편집할 필요가 없어요.

---

## 4단계. 기기에 복사

USB-C로 기기를 컴퓨터에 연결한 뒤, **터미널(맥/리눅스) 또는 PowerShell(윈도우)** 을 열고
압축 푼 폴더가 있는 위치에서 아래 명령을 실행하세요. (한글 버전 기준)

```sh
scp -O -r riddle-move-ko root@10.11.99.1:/home/root/xovi/exthome/appload/
```

- 비밀번호를 물어보면, 기기에서
  **설정 → 도움말 → 저작권 및 라이선스 → General Information → GPLv3 Compliance**
  에 적힌 비밀번호를 입력하세요. (화면에 안 보여도 입력되는 중이에요)
- `10.11.99.1` 은 USB로 연결했을 때의 기본 주소입니다.

---

## 5단계. 실행

1. 기기에서 **AppLoad** 를 엽니다.
2. **Reload** 를 눌러 목록을 새로고침해요.
3. **The Diary (한글)** 을 탭하면 실행됩니다.
4. 펜으로 한 문장 쓰고 **펜을 잠깐 떼면**, 페이지가 글씨를 삼키고 톰이 손글씨로 답장을 써요. ✍️

---

## ✋ 사용법 (제스처)

| 동작 | 방법 |
|------|------|
| 질문하기 | 펜으로 쓰고 잠깐 멈추기 |
| 지우기 | 펜(마커)을 뒤집어서 문지르기 |
| 도움말 보기 | 큰 물음표 **?** 하나 그리기 |
| 나가기 | 다섯 손가락으로 화면 탭 |
| 잠자기 | 전원 버튼 |

## 🛠️ 문제 해결

- **답장이 안 와요** → 대부분 API 키 문제예요. `oracle.env` 의 키가 정확한지, OpenAI 계정에 결제/크레딧이 있는지 확인하세요.
- **화면이 멈췄어요 / 못 빠져나와요** → 컴퓨터 터미널에서
  `ssh root@10.11.99.1` 접속 후 `systemctl start xochitl` 입력하면 원래 화면으로 돌아옵니다.
- **AppLoad 목록에 안 보여요** → 폴더를 `.../appload/` **바로 아래**에 복사했는지 확인하고 다시 **Reload**.
- **영어로 썼는데 한글로 답해요** → 최신 버전(v0.1-move 이상)인지 확인하세요. 최신 버전은 쓴 언어를 따라갑니다.
